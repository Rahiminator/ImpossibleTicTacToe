# Impossible Tic Tac Toe (Unbeatable Computer Player)

A simple Tic Tac Toe game built in Python using Tkinter.  
The computer opponent follows optimal strategy, meaning the player can never win.

---

## Requirements

- Python 3.x
- Tkinter (included with most Python installations)

Check your Python version:

```bash
python3 --version
```
---

## Running the Game

Run the main game file:

```bash
python3 tic_tac_toe.py
```

---

## Running Tests

Run the unit test file:

```bash
python3 test_tic_tac_toe.py
```

---

## Expected Test Output

All tests should pass:

